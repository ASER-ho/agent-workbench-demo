# Agent Workbench

> AI 帮你写完了项目。但它不会帮你检查能不能安全发出去。

[**👉 点此在线体验**](https://aser-ho.github.io/agent-workbench-demo/)

---

## 这是什么

一个帮 AI 辅助开发项目做**交付前检查、脱敏、打包、生成收据**的本地工作台。

你在 AI 的帮助下做了个项目，想发给同事、交作业、上传 Demo。在点"发送"之前——项目里有没有密钥？有没有本地路径？有没有缓存和日志？

Agent Workbench 模拟了从检查到交付的完整闭环。

## 怎么用

1. 打开页面，先**贴一段带密钥的文本试试**——这是真实可用的脱敏功能
2. 点「体验完整流程」一键跑通：保存档案 → 环境扫描 → 密钥脱敏 → 分享打包 → 下载收据
3. 或在左侧导航逐步操作每个模块

## 核心功能

- **即时脱敏**：贴文本进去，密钥/路径/token 自动遮蔽（真实可用）
- **密钥守卫**：模拟文件扫描 + 脱敏前后对比
- **交付前检查**：逐项核对，没完成脱敏不允许打包
- **项目对比**：原始项目 vs 安全分享包左右对照
- **收件人视角**：切换到对方视角看最终收到什么
- **工作收据**：生成可复制、可下载的交付说明

## 本地使用

下载 `index.html`，双击在浏览器中打开即可。无需安装任何东西。

## 技术说明

纯单文件 HTML + CSS + JS，零外部依赖。所有数据为模拟演示数据，不读取真实系统。
